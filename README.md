# account-api
![deploy](https://github.com/Reckue/account-api/workflows/deploy/badge.svg)

Service for user authorization.

## Stack
- JDK 11
- Gradle
- Spring Boot
- Spring Data
- Spring Security
- MongoDB
- Lombok
- Swagger
- Devtools
- Apache Commons
